**주요 성과**

- 300+ DB 인스턴스를 실시간 감시해야 하는 레거시 관제 환경에서 Vue 대시보드를 React + TypeScript 기반으로 마이그레이션하고 중앙 폴링 아키텍처를 도입해 장애 인지 시간을 70% 단축했습니다(기준: 동일 알람 시나리오의 알림 노출까지 평균 시간 10초→3초).
- 대량 행 렌더링으로 스크롤·리사이즈 지연이 발생하던 데이터 그리드를 TanStack Table 기반 가상화 구조로 재설계해 DOM 노드를 90% 줄이고 리사이즈 처리 속도를 44배 개선했습니다(기준: 동일 데이터셋·동일 브라우저 조건의 내부 벤치마크 비교).
- 시각화 유형별로 필드 의존성이 복잡한 차세대 폼 요구사항에 어댑터 패턴과 Zustand 스토어 팩토리를 결합해 상태를 컴포넌트 단위로 지역화했고, 신규 폼 추가 시 핵심 로직 수정 없이 확장 가능하도록 보일러플레이트를 70% 절감했습니다(기준: 동일 기능 추가 시 평균 코드 라인 수 비교).
- 폐쇄망 환경에서 배포 결과 추적과 개발 환경 셋업이 느려지던 문제를 해결하기 위해 TypeScript 기반 CI/CD 리포트 뷰어와 온보딩 CLI를 자동화해 레포트 접근 시간을 97% 단축하고 신규 입사자 환경 구성을 수 시간에서 5분으로 줄였습니다(기준: 사내 표준 온보딩 체크리스트 완료 시간 측정).
